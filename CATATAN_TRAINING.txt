Pelatihan tesseract

Ah, baiklah! Anda ingin fokus pada **kode untuk melatih Tesseract OCR** menggunakan Python. Berikut adalah panduan dan skrip dasar untuk memulai pelatihan Tesseract agar mengenali teks seperti di gambar Anda (dengan karakter Latin dan diakritik seperti **ā, ö, ê**).

---

### **1. Prasyarat**
Sebelum menjalankan kode, pastikan Anda telah:
1. **Menginstal Tesseract**:
   ```bash
   brew install tesseract
   ```
2. **Menginstal Python** dan pustaka pendukung:
   ```bash
   pip install pillow
   ```

---

### **2. Langkah Utama**
Pelatihan Tesseract memerlukan data berupa gambar teks dan transkripsinya. Kita akan menggunakan Python untuk membuat data ini.

#### **Skrip Python untuk Mempersiapkan Data Pelatihan**

```python
from PIL import Image, ImageDraw, ImageFont
import os

# Direktori output untuk menyimpan gambar dan transkrip
output_dir = "training_data"
os.makedirs(output_dir, exist_ok=True)

# Konfigurasi font dan ukuran
font_path = "/System/Library/Fonts/Supplemental/Times New Roman.ttf"  # Ganti dengan font diakritik yang sesuai
font_size = 32
font = ImageFont.truetype(font_path, font_size)

# Contoh teks untuk pelatihan
training_text = [
    "Hana sira ratu dibya rênön",
    "praśāsta riņ rāt musuh nira praṇata",
    "jaya paṇḍita riņ aji kabeh",
    "sañ Daśaratha nāma tamoli"
]

# Buat gambar untuk setiap baris teks
for idx, line in enumerate(training_text):
    image_width = 800
    image_height = 100
    image = Image.new("L", (image_width, image_height), color=255)  # Gambar hitam-putih
    draw = ImageDraw.Draw(image)
    
    # Tulis teks ke dalam gambar
    text_position = (10, 10)
    draw.text(text_position, line, font=font, fill=0)

    # Simpan gambar dan transkripsi
    image.save(f"{output_dir}/line_{idx + 1}.tif")
    with open(f"{output_dir}/line_{idx + 1}.gt.txt", "w", encoding="utf-8") as f:
        f.write(line)

print(f"Data pelatihan disimpan di: {output_dir}")
```

Kode ini akan:
1. Membuat gambar TIFF dari teks dengan diakritik.
2. Menyimpan file `.tif` dan transkripnya (`.gt.txt`) di direktori `training_data`.

---

### **3. Melanjutkan Pelatihan dengan Tesseract**
Setelah data pelatihan siap, gunakan Tesseract untuk melanjutkan pelatihan.

#### **Langkah Pelatihan dengan Perintah Terminal**
1. Konversi gambar menjadi file pelatihan:
   ```bash
   tesseract training_data/line_1.tif training_data/line_1 lstm.train
   ```

2. Gabungkan semua file `.lstmf` ke dalam satu daftar:
   ```bash
   echo training_data/*.lstmf > training_data/all-lstmf.txt
   ```

3. Lakukan pelatihan:
   ```bash
   lstmtraining --continue_from /path/to/baseline.traineddata \
   --model_output my_model \
   --traineddata /path/to/langdata \
   --train_listfile training_data/all-lstmf.txt \
   --max_iterations 1000
   ```

4. Gabungkan model:
   ```bash
   combine_tessdata -o my_model.traineddata my_model
   ```

---

### **4. Output Akhir**
- Model baru (`my_model.traineddata`) akan dihasilkan.
- File ini dapat Anda gunakan untuk OCR dengan bahasa yang telah Anda latih:
  ```bash
  tesseract test-image.tif output -l my_model
  ```

Tentu! Tesseract OCR sangat fleksibel, dan Anda dapat menggunakannya di macOS serta melatihnya untuk mengenali karakter khusus seperti `ā ī ū ṅ ñ è ĕ ö`. Berikut adalah langkah-langkahnya:

---

### **1. Instalasi Tesseract di macOS**
Anda dapat menginstal Tesseract menggunakan **Homebrew**:

1. Pastikan Homebrew sudah diinstal. Jika belum, jalankan:
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. Instal Tesseract dengan Homebrew:
   ```bash
   brew install tesseract
   ```

3. Untuk memverifikasi instalasi:
   ```bash
   tesseract --version
   ```

---

### **2. Melatih Tesseract untuk Karakter Khusus**

Untuk melatih Tesseract agar mengenali karakter khusus seperti `ā ī ū ṅ ñ è ĕ ö`, Anda perlu membuat data pelatihan kustom. Berikut langkah-langkahnya:

#### **a. Siapkan Data Pelatihan**
1. **Buat dataset teks** yang berisi semua karakter yang ingin dikenali. Misalnya:
   ```
   ā ī ū ṅ ñ è ĕ ö
   ```

2. Gunakan alat seperti `jTessBoxEditor` atau `Tesseract Training Tool` untuk membuat file **box** (berisi koordinat setiap karakter) yang sesuai dengan data gambar.

---

#### **b. Proses Pelatihan**
1. **Buat gambar pelatihan**: Gunakan alat seperti Photoshop atau GIMP untuk membuat gambar teks (contoh font dan ukuran) berisi karakter yang Anda inginkan.

2. **Hasilkan file box**:
   - Jalankan perintah untuk membuat file box:
     ```bash
     tesseract image.png output_name box.train
     ```

3. **Edit file box**:
   - Gunakan editor teks untuk memastikan setiap karakter diidentifikasi dengan benar dalam file `.box`.

4. **Kompilasi pelatihan**:
   - Jalankan perintah berikut untuk memulai proses pelatihan:
     ```bash
     tesseract image.png output_name lstm.train
     ```

5. **Gabungkan hasil pelatihan**:
   - Gunakan `combine_tessdata` untuk menggabungkan hasil pelatihan ke dalam file `.traineddata`:
     ```bash
     combine_tessdata -o custom_font.traineddata
     ```

6. **Pindahkan file `.traineddata` ke folder Tesseract**:
   ```bash
   mv custom_font.traineddata /usr/local/share/tessdata/
   ```

---

### **3. Menggunakan Model Pelatihan Kustom**
Sekarang Anda dapat menggunakan model pelatihan kustom dengan perintah seperti ini:
```bash
tesseract input_image.png output_text --oem 1 -l custom_font
```

- Pastikan gambar pelatihan memiliki kualitas tinggi dan karakter tidak saling bertumpuk.
